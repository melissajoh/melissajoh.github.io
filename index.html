An adventure game I made as a last project for my Object-oriented web technologies course. You play by reading and writing commands like "go north" (in Swedish). The game functionality relies a lot on an ORM database, classes and controllers along with an API. The database for example has a room and an items table to check where we are able to go, if an item can be or has been picked up and so on. The API has routes for the backpack, items and rooms to keep track. The images are made by me. The game is very simple and could definitely be developed further, but is an example of a coding project I have done in the past. Can be opened by cloning this repo and navigating to "Projekt".
